# Sistema de Convocatorias CSSI

Herramienta digital para la gestión de convocatorias de financiamiento del Centro de Salud Santa Inés UCAB.

## Funcionalidades

- Registro de convocatorias
- Seguimiento de estatus
- Visualización de fechas límite
- Asignación de responsables
- Cálculo de métricas clave
- Priorización por probabilidad de éxito

## Objetivo

Centralizar y optimizar la gestión de oportunidades de financiamiento, facilitando la toma de decisiones estratégicas dentro del CSSI.

## Tecnologías

- React
- Vercel (deployment)
- Futuro: Supabase (base de datos)

## Estado del proyecto

En desarrollo – primera versión funcional
